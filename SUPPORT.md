# Support

## Getting Help

If you encounter issues or have questions:

- Open an issue
- Start a discussion
- Contact the maintainers

## Reporting Bugs

Please include:

- Operating system
- Software version
- Steps to reproduce
- Expected behavior
- Actual behavior

## Feature Requests

Suggestions are welcome through GitHub Issues or Discussions.

## Security Issues

Please do not report security vulnerabilities publicly.
Refer to `SECURITY.md` for responsible disclosure.
